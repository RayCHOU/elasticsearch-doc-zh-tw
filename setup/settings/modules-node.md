# [Node](https://www.elastic.co/guide/en/elasticsearch/reference/current/modules-node.html)

任何時候啟動 Elasticsearch 的 instance，都會啟動一個 node。  
連接在一起的 nodes 集合稱為 cluster。  
如果您正在運行 Elasticsearch 的單個節點，那麼您將擁有一個由一個節點組成的集群。

(略)
